Instructions

  1. Begin by selecting whether to study the fly or yeast. You can switch which creature you are viewing via the “organisms” on the lefthand side bar
  2. Upon selecting your organism, you will view the main visualization page. There are several components to this page:
    • The Left Side Bar
    • The Network Visualization
    • The "Update Inputs" Button (top right corner)
    • The Gene Information
    • The GO Network
  3. The Left Side Bar:
    This side bar contains links to our organism selection via the "organisms" link, to the instructions via the "Instructions" link, to the Process Book via the "Process Book" link, and will eventually allow for a user to export their visualization results via the "Export Results" link. This sidebar also contains notable reference materials and contact information to the Brent Lab.
  4. The Network Visualization:
    Once you select an organism, a TF (Transcription Factor) network will appear by default. The blue node is a Transcription Factor (read the tf story!), the red nodes are the genes that this Transcription Factor regulates.
  5. The "Update Inputs" Button
    Next, please click the button "Update Inputs" on the top right hand portion of the screen. A side panel will appear. At the top of the side panel is a section to filter the network with edge weights. The values represented in this normalized distribution represent the strength of evidence that the selected Transcription Factor controls a given gene. Therefore, selecting a range within this graph will return all genes regulated by the TF where the strength of evidence is between the given range.
    Use the brush to select a portion of the curve on the far righthand side. If the network is large (lots of red nodes), select regions further to the right side of the distribution and click "Update Weights". Too many nodes still? Select even less of the curve and update.
    Next, try entering a transcription factor or a gene list (The gene list option is currently not implemented).If you are a fly biologist, try entering FBgn0016917. It has a lot of targets -- you may want to increase your minimum threshold!
  6. The Gene Information:
    When you click a gene (all nodes are genes, but the blue one is special because it is a regulator of the red genes), information is populated in the "Gene Detail". This is a place holder at this point, though some information will be populated.
  7. The GO Network
    If there isn't much interesting happening on the Gene Ontology (GO) graph below the node/edge graph, go back to the landing page by clicking "organism" on the left side panel, re-enter the organism page, to see what a new network looks like. Do this until you get more than a handful of dots or enter a TF id in the right side panel (As mentioned before, FBgn0016917 is a good place to start). Different network clusters return more significant hits (see below) than others.
    GO is one of computational biology's methods of beginning to figure out what genes do and how they relate to one another. The graph is created by sending the target gene list to an external database, which returns GO annotation and a significance score, among other things. The x-axis are different categories that the genes fall into -- for example, GO:BP is a "biological process" while GO:MF is a "molecular function". The dots are (size and height) ranked by their significance. Only annotations that have an adjusted p-value greater than .05 are displayed, and the scale is -log10, so the higher the better.
    This graph does not have its interactivity yet. Please see the process book for a drawing of what it will do. Eventually, there will be functionality to allow the user to group the genes in various different ways (eg, by whether or not there is experimental support for the TF/target edge, or by the gene function).
